# 🚀 DEVQUIZ – Full Stack Quiz Platform

## 📌 Overview

DEVQUIZ is a full-stack web application designed to provide an interactive quiz experience for users. It allows users to attempt quizzes, track performance, and manage content efficiently through a modern UI and scalable backend.

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS / Tailwind

### Backend

* Node.js
* Express.js
* MongoDB

---

## ✨ Features

* 🧠 Attempt quizzes with real-time interaction
* 📊 Track user performance
* 🔐 Authentication & authorization (if implemented)
* ⚡ Fast and responsive UI
* 📦 Scalable backend API

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/mohityadav1407/DEVQUIZ.git
cd DEVQUIZ
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
